Ayuda
¿Cómo usar esta herramienta?
  1. Filtra por cuatrimestre para reducir la cantidad de materias mostradas
  2. Revisa la matriz de asignaciones:
    • Las filas representan profesores de tu carrera ({{ carrera.nombre }})
    • Las columnas muestran las materias con su código y nombre completo
    • Checkbox verde marcado = materia ya asignada
    • Checkbox vacío = materia sin asignar
  3. Realiza cambios:
    • Para asignar: Marca el checkbox vacío
    • Para desasignar: Desmarca el checkbox verde
    • Los checkboxes desmarcados se mostrarán en rojo para indicar desasignación
  4. Haz clic en "Guardar Cambios" para aplicar las modificaciones
Consejos:
  • "Seleccionar Todas": Marca todos los checkboxes (asigna todas las materias a todos los profesores)
  • "Limpiar Selección": Restaura al estado original (cancela todos los cambios)
  • Códigos de color: Verde = Asignada, Azul = Nueva asignación, Rojo = Se va a desasignar
