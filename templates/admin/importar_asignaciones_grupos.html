Importar Asignaciones de Materias a Grupos

Instrucciones
  1. Descarga la plantilla CSV o exporta las asignaciones actuales
  2. Edita el archivo con tus asignaciones (usa Excel, LibreOffice o editor de texto)
  3. Guarda el archivo como CSV (separado por comas)
  4. Sube el archivo usando el formulario abajo
Formato del Archivo CSV

El archivo debe tener estas columnas:

  • grupo_codigo: Código del grupo (ej: 1MSC1, 2MVE3)
  • materia_codigo: Código de la materia (ej: MAT101, FIS201)

Ejemplo:

grupo_codigo,materia_codigo
1MSC1,MAT101
1MSC1,FIS101
1MSC1,PROG101
2MSC3,BD201
2MSC3,WEB201
Importante:
  • Los códigos de grupos y materias deben existir en el sistema
  • La materia debe pertenecer a la misma carrera y cuatrimestre que el grupo
  • Las asignaciones se agregan a las existentes (no las reemplazan)

Tips y Recomendaciones
  • Validación automática: El sistema validará que los códigos existan
  • Errores detallados: Si hay errores, se te mostrará qué líneas tienen problemas
  • Codificación: Guarda el archivo en UTF-8 para evitar problemas con acentos
  • Sin duplicados: Si un grupo ya tiene una materia asignada, se ignorará
  • Procesamiento por lotes: Puedes subir cientos de asignaciones a la vez
